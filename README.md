# Description
Velocity plugin designed to bridge communication with the backend via RabbitMQ. This project serves as a robust example for developers looking to implement custom authentication, skin management, or nickname systems within a Velocity proxy.

* Key features include:
  * Remote Session Management: Kick players on demand via their authorization key or username.
  * Connection Handling: Accept or reject player login requests based on client-side mod authentication.
  * GameProfile Modification: Intercept GameProfileRequest to dynamically inject custom skins, UUIDs, and nicknames.
